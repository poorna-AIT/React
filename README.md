## Getting Started

1. Clone Repository
```
git clone https://github.com/themewagon/docgenAI.git
```
2. Install Dependencies
```
npm i
```
3. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
